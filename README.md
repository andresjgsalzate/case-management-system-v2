# Case Management System

Sistema de gestión de casos de soporte técnico.

**Stack:** FastAPI · Next.js 14 · PostgreSQL 16 · Redis 7

**Arquitectura:** Monolito Modular + Event-Driven + BFF

## Fases de desarrollo

| Fase | Módulo |
|------|--------|
| 0 | Fundación (monorepo, config, Event Bus) |
| 1 | Identidad — Auth, Users, Roles, Teams |
| 2 | Núcleo de Casos |
| 3 | Clasificación + SLA |
| 4 | Colaboración (Chat, Notes, TODOs, Timer) |
| 5 | Disposiciones + Métricas + Búsqueda |
| 6 | Base de Conocimiento |
| 7 | Notificaciones + Auditoría + Automatización |
| 8 | Frontend completo |
| 9 | Despliegue (Linux + Windows) |
